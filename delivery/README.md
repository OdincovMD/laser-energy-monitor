# Delivery Folder

Эта папка предназначена для подготовки клиентской поставки.

Рекомендуемый состав:

- `app/` - release-сборка приложения `LaserEnergyMonitor.Wpf`
- `docs/` - клиентские инструкции из каталога `docs/`

Важные замечания:

- не складывайте сюда локальные тестовые или устаревшие выкладки без необходимости;
- временные папки вида `LaserEnergyMonitor-*` считаются промежуточными артефактами и не должны использоваться как источник истины;
- перед передачей заказчику стоит заново собрать актуальную `Release|x86` версию и положить рядом свежие инструкции.
